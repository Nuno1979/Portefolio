
# 🎧 Music Industry Analysis (Power BI)

## 📊 Project Overview

This project analyzes music industry data to uncover trends in artist popularity, album performance, and song success.

The analysis focuses on three main areas:

* Albums
* Artists
* Songs

## 🎯 Objective

The goal is to identify patterns that define successful music over time.

## 🛠 Tools Used

* Power BI
* DAX
* Data Modeling

## 📁 Dataset

Data sourced from Kaggle, including albums, artists, and songs.

## 📌 Status

Project in progress – currently improving insights and visuals.
